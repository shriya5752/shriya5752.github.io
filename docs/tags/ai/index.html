§ 1 — Traditional Programming vs ML
Traditional Programming Data + Rules → Output You write the if/else. Computer executes.
Machine Learning Data + Output → Rules You show examples. Computer finds the rules.
Deep Learning Raw Data + Output → Complex Rules Learns layered representations automatically.
The Three Things ML Needs
📊 Data Examples of inputs and (usually) outputs. The raw material of learning.
🧮 A Model A mathematical function that maps inputs to outputs — its parameters are learned.
📉 A Loss Function A way to measure how wrong the model is — so it knows which direction to improve.
